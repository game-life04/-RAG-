# -RAG-
中医智能诊疗是医疗领域在科技高速发展的新时代传统医学与现代技术融合的态势。二者的结合为中医（Traditional Chinese Medicine，TCM）带来了更契合社会需求的发展方向。研究构建了基于大语言模型（Large Language Model，LLM）与检索增强生成（Retrieval-Augmented Generation，RAG）的中医智能辅助诊疗系统。该系统融合了知识驱动与数据驱动的推理模式，实现了在诊疗过程中结合中医知识图谱进行结构化推理，并运用 RAG 技术动态检索权威医学文献，同时利用语义索引优化知识检索。该系统在智能问诊、证候辨识、方剂推荐等方面展现出显著的优势，为中医智能化发展提供了更全面的技术支持。

**3.1：GNN(Graph Neural Network（图神经网络）是一种专门用来处理“图结构数据”的神经网络****
#### GNN 到底在“学什么”
tip:一个节点的表示=自己的信息+邻居的信息——>Message Passing信息传输
1：Node(节点)+Edge（边）+Aggregation（聚合）基本组成原理
2：Step 1：初始化——>hv(0)​，Step 2：从邻居收集信息——>mv(k)​=AGGREGATE({hu(k−1)​∣u∈N(v)})，Step 3：更新自己——>hv(k)​=σ(W⋅[hv(k−1)​∣∣mv(k)​])


**3.2：BM25+Dense Retriever(传统关键词检索(精确)+向量语义检索(理解含义)的混合检索方案)**
1：==**BM25**==：本质是基于关键词匹配的检索算法——>专有名词快准稳，但是没有发散思维
2：==**Dense Retriever**==：基于向量相似度的语义检索——>懂语义，懂同义，但是对专有名词和编号不敏感
3：使用方式 1：并行召回（最常用）——>方式 2：BM25 → Dense Re-rank——>方式 3：加权融合（科研常用）->Score=α⋅BM25+(1−α)⋅Dense(总和为1)

**3.3：Transformer+BiLSTM+CRF(是一种用于 序列标注任务（如 NER） 的组合模型)——>核心应用：命名实体识别（NER）**
- Transformer 负责 **上下文语义建模**
    
- BiLSTM 负责 **顺序依赖建模**
    
- CRF 负责 **全局标签约束解码**
