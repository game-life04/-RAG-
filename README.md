# -RAG-
中医智能诊疗是医疗领域在科技高速发展的新时代传统医学与现代技术融合的态势。二者的结合为中医（Traditional Chinese Medicine，TCM）带来了更契合社会需求的发展方向。研究构建了基于大语言模型（Large Language Model，LLM）与检索增强生成（Retrieval-Augmented Generation，RAG）的中医智能辅助诊疗系统。该系统融合了知识驱动与数据驱动的推理模式，实现了在诊疗过程中结合中医知识图谱进行结构化推理，并运用 RAG 技术动态检索权威医学文献，同时利用语义索引优化知识检索。该系统在智能问诊、证候辨识、方剂推荐等方面展现出显著的优势，为中医智能化发展提供了更全面的技术支持。
